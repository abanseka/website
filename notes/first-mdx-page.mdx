### Blogging With NEXT.JS & MDX 🔥🔥

```javascript
import Head from "next/head";
import { MainContainer } from "@components/styled/Main";

const index = () => {
  return (
    <>
      <Head>
        <title>Journal</title>
        <meta name="description" content="abanseka's journal" />
        <link rel="icon" href="/branch_favicon.png" />
      </Head>

      <MainContainer>
        <h1>Journal</h1>
      </MainContainer>
    </>
  );
};

export default index;
```

## Getting Started 🔥🔥

to get started, you need to install the [next.js](https://nextjs.org/docs/getting-started)
and [mdx](https://github.com/mdx-js/mdx) packages.

- [next.js](https://nextjs.org/docs/installation)
- [mdx](https://github.com/mdx-js/mdx)
- [npx mdx-cli](https://github.com/mdx-js/mdx/blob/main/packages/mdx-cli/README.mdx)
